# Space Rangers 2 Screen Reader

Инструмент для автоматического захвата экрана, распознавания текста с помощью Google Gemini API и озвучивания текста (Text-to-Speech) для игры "Космические рейнджеры 2".

## Описание

Данное приложение предназначено для помощи в чтении текстовых квестов и диалогов в игре. При нажатии горячей клавиши программа делает скриншот активного окна, отправляет его в API Google Gemini для извлечения сюжетного текста и озвучивает результат.

## Функциональность

*   **Глобальные горячие клавиши**: Управление захватом и чтением из любого окна.
*   **ИИ Распознавание**: Использование модели Gemini Flash для точного выделения текста квеста и игнорирования элементов интерфейса.
*   **Синтез речи**: Озвучивание текста с использованием системного движка TTS.

## Требования

*   OS: Windows
*   Python 3.8+
*   Ключ API Google Gemini

## Установка

1.  Установите необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```
2.  Настройте переменные окружения:
    *   Переименуйте файл `.env.example` в `.env`.
    *   Укажите ваш API ключ в файле `.env`:
        ```
        GEMINI_API_KEY=ваш_ключ
        ```

## Использование

1.  Запустите приложение:
    ```bash
    python main.py
    ```
2.  В игре нажмите сочетание клавиш **Ctrl + L** для захвата экрана и озвучивания текста.
3.  Для завершения работы нажмите **Ctrl + Shift + Q** или **Ctrl + C** в консоли.

## Структура проекта

*   `main.py`: Точка входа, обработка горячих клавиш.
*   `capture.py`: Модуль захвата изображения с экрана.
*   `ai_service.py`: Модуль взаимодействия с API Google Gemini.
*   `tts_service.py`: Модуль синтеза речи.

## Лицензия

Распространяется свободно для личного использования.
